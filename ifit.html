iFit Tone Generator

By: Michael K. Pellegrino

Version 1.04 - 2022 01 25

This program will help you create an iFit workout audio track to coincide with a music playlist.

The program uses wav.js by Kevin Phillips.

  1. Enter a time (minutes and seconds), resistance/incline, and speed if needed (10 - 100 in increments of 10) for your workout.

    Ellipticals don't use the "Speed" signal, so if you're using one, you can check the "Elliptical" checkbox. The speed value won't be even looked at.

  2. Click "Add to Workout"
  3. Enter a filename for your wav file.
  4. Enter an offset time to start controlling the iFit

    5. When I use the elliptical, I start the music first. Then, on a specific cue from the music (like a cymbal crash, or when a certain note plays), I start the iFit signal sequence. That way I know it'll be synchronized the way that I want it to be. For the offset you would enter in the time at which that cue happens in the music. For instance, I usually work out to the Dark Side of the Moon. In my playlist, the first guitar strum is at 1 minute, sixteen seconds into the playlist, that's when I start the signals... so that's my offset... 1min 16sec

  5. Click "Download WAV file" to get the audio file.
  6. Then burn it to a CD.
